How to Use S3 Colors

There are 3 simple steps to use S3-Colors in your project.

  1. Call s3-colors.css in the head tag <link rel="stylesheet" type="text/css" href="css/s3-colors.css">
  2. Call color name in your element for text <span class="ColorName">your text</span>
  3. Call color name in your element for background class="bg-ColorName"
  4. Color Name are listed below with their details
