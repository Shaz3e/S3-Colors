How to Use S3 Colors

There are 3 simple steps to use S3-Colors in your project.

Download S3-Color.css
  1. Download S3 Color from the link above and call s3-colors.css in the head tag <link rel="stylesheet" type="text/css" href="your/path/to/s3-colors.css"> CDN Version: <link rel="stylesheet" type="text/css" href="http://cdnjs.cloudflare.com/ajax/libs/s3colors/1.0/s3-colors.css"> CDN Minified Version: <link rel="stylesheet" type="text/css" href="http://cdnjs.cloudflare.com/ajax/libs/s3colors/1.0/s3-colors.min.css">
  2. Call color name in your element for text <span class="ColorName">your text</span>
  3. Call color name in your element for background class="bg-ColorName"
  4. Color Name are listed below with their details
